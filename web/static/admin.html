Desktop Setup Instructions

  1. Clone the repo on your desktop machine
  2. Copy .env.example to .env and add your API keys
  3. Install deps: pip install -r requirements.txt
  4. Run: ./run_benchmark.sh
# .env example OPENAI_API_KEY=sk-... ANTHROPIC_API_KEY=sk-ant-... GOOGLE_API_KEY=AIza... DEEPSEEK_API_KEY=sk-... UPLOAD_API_KEY=<from admin panel> LLMBENCH_UPLOAD_URL=https://simonmccallum.org.nz/bench
